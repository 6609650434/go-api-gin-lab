# Student API with Gin

A simple RESTful API built with Go and Gin framework.

---

## How to Run

1. Install dependencies
go mod tidy

2. Run the application
go run main.go

Server will start at:
http://localhost:8080

---

## Available API Endpoints

GET /students  
GET /students/:id  
POST /students  
PUT /students/:id  
DELETE /students/:id
